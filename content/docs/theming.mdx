Copy the following CSS variables into your global stylesheet.

This theme includes both light and dark color schemes. The `.dark` class will automatically apply the dark mode when present.

> Note: The variables use OKLCH color values, which are supported by all modern browsers. If you want to use them in Tailwind class names, they have been mapped via `@theme inline`.
