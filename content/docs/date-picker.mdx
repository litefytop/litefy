## API Reference

### DatePicker

The date picker input component.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `type` | `'date' \| 'time' \| 'datetime-local' \| 'month' \| 'week'` | `'date'` | The picker type |
| `placeholder` | `string` | - | Placeholder text shown when no value is selected |
| `value` | `string` | - | Controlled value |
| `invalid` | `boolean` | - | Whether the input is in an invalid state |
| `disabled` | `boolean` | - | Whether the input is disabled |
| `className` | `ClassNameValue` | - | Custom CSS class. Supports strings, arrays, objects, booleans, and more. |
| `ref` | `React.Ref<HTMLInputElement>` | - | Ref forwarded to the input element |
