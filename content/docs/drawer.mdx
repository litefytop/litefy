## API Reference

### Drawer

A slide-in panel component with focus management and keyboard navigation.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `ClassNameValue` | - | Custom CSS class |
| `placement` | `'left' \| 'right' \| 'top' \| 'bottom'` | `'right'` | Position of the drawer |
| `onClose` | `() => void` | - | Callback when drawer is closed |
| `onOpen` | `() => void` | - | Callback when drawer is opened |
| `aria-label` | `string` | - | Accessibility label |
| `aria-labelledby` | `string` | - | ID of the element labeling the drawer |

### DrawerRef

The ref object returned by the Drawer component.

| Method | Description |
|--------|-------------|
| `show()` | Open the drawer |
| `close()` | Close the drawer |
