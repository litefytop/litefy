## API Reference

### Drawer

A slide-in panel component with focus management and keyboard navigation.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `ClassNameValue` | - | Custom CSS class. Supports strings, arrays, objects, booleans, and more. |
| `placement` | `'left' \| 'right' \| 'top' \| 'bottom'` | `'right'` | Position of the drawer |
| `onClose` | `() => void` | - | Callback when drawer is closed |
| `ref` | `React.Ref<HTMLDialogElement>` | - | Ref forwarded to the native dialog element |

### slotProps

| Slot | Type | Description |
|------|------|-------------|
| `content` | `HTMLAttrs<Omit<React.ComponentProps<'div'>, 'style'>>` | Props for the inner content panel |
