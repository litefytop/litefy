## Introduction

Litefy UI is a lightweight component library with almost no external dependencies. Every component is copy-paste friendly and ships with a CLI for faster setup.

## Installation

<Tabs items={['CLI', 'Manual']}>

<Tab value="CLI">

The fastest way to add a component is through the CLI. It drops the source file into your project so you own the code.

<PackageManagerTabs command="litefy add button" />

</Tab>

<Tab value="Manual">

Copy the component source directly from the docs and paste it into your project. Each component page exposes the full source.

<SourceCode component="button" />

</Tab>

</Tabs>
