## API 参考

### Carousel

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `React.ReactNode` | - | 轮播内容 |
| `activeIndex` | `number` | - | 当前活动轮播索引 |
| `onChange` | `(index: number) => void` | `undefined` | 轮播切换时的回调 |
| `autoPlay` | `boolean` | `false` | 启用自动轮播 |
| `autoPlayInterval` | `number` | `3000` | 轮播间隔时间（毫秒） |
| `loop` | `boolean` | `false` | 启用无限循环 |
| `slotProps` | `object` | `undefined` | 内部插槽的自定义属性 |

### slotProps

| Slot | Type | Description |
|------|------|-------------|
| `wrapper` | `React.ComponentProps<"section">` | 包装器 div 的属性 |
| `list` | `React.ComponentProps<"ul">` | 列表元素的属性 |
| `slide` | `React.ComponentProps<"li">` | 每个轮播项 div 的属性 |
