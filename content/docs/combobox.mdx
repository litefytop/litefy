## API Reference

### Combobox

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | `undefined` | Controlled value |
| `defaultValue` | `string` | `""` | Default value (uncontrolled) |
| `onChange` | `(e: ComboboxChangeEvent) => void` | `undefined` | Callback when value changes |
| `onSelect` | `(option: string) => void` | `undefined` | Callback when option is selected |
| `options` | `string[] \| ((keyword: string) => Promise<string[]>)` | `undefined` | Static options or async fetch function |
| `placeholder` | `string` | `"Search or type..."` | Input placeholder |
| `disabled` | `boolean` | `false` | Disable the combobox |
| `invalid` | `boolean` | `false` | Mark as invalid |
| `clearable` | `boolean` | `true` | Show clear button |
| `debounceMs` | `number` | `300` | Debounce delay for async search |
| `skeleton` | `React.ReactNode` | `undefined` | Custom loading skeleton |
| `optionHeight` | `number` | `36` | Height of each option in pixels |
| `overscan` | `number` | `2` | Number of extra options to render outside viewport |

### slotProps

| Slot | Type | Description |
|------|------|-------------|
| `container` | `React.HTMLAttributes<HTMLDivElement>` | Props for the container div |
| `input` | `React.InputHTMLAttributes<HTMLInputElement>` | Props for the input |
| `list` | `React.HTMLAttributes<HTMLUListElement>` | Props for the dropdown list |
| `option` | `React.HTMLAttributes<HTMLLIElement>` | Props for each option |
| `clearButton` | `React.ButtonHTMLAttributes<HTMLButtonElement>` | Props for the clear button |
