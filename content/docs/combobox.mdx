## API Reference

### Combobox

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | - | Controlled value |
| `defaultValue` | `string` | - | Default value (uncontrolled) |
| `onChange` | `(e: ComboboxChangeEvent) => void` | - | Callback when value changes |
| `onBlur` | `React.FocusEventHandler<HTMLInputElement>` | - | Callback when input loses focus |
| `onSelect` | `(option: string) => void` | - | Callback when option is selected |
| `options` | `string[] \| `<br/> `({page: number; size: number; keyword: string}) => Promise<{list: string[]; total: number}>` | - | Static options or async fetch function with pagination |
| `placeholder` | `string` | - | Input placeholder |
| `disabled` | `boolean` | `false` | Disable the combobox |
| `invalid` | `boolean` | `false` | Mark as invalid |
| `debounceMs` | `number` | `300` | Debounce delay for async search |
| `skeleton` | `React.ReactNode` | - | Custom loading skeleton |
| `pageSize` | `number` | `20` | Number of items per page for pagination |

### slotProps

| Slot | Type | Description |
|------|------|-------------|
| `container` | `HTMLAttrs<Omit<React.ComponentProps<"div">, "ref">>` | Props for the container div |
| `input` | `HTMLAttrs<Omit<React.ComponentProps<"input">, "disabled" \| "placeholder" \| "ref" \| "value" \| "defaultValue">>` | Props for the input |
| `list` | `HTMLAttrs<Omit<React.ComponentProps<"ul">, "ref">>` | Props for the dropdown list |
| `option` | `HTMLAttrs<React.ComponentProps<"li">>` | Props for each option |
