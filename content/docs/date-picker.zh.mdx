## API 参考

### DatePicker

日期选择器输入组件。

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `type` | `'date' \| 'time' \| 'datetime-local' \| 'month' \| 'week'` | `'date'` | 选择器类型 |
| `placeholder` | `string` | - | 未选择值时显示的占位文本 |
| `value` | `string` | - | 受控值 |
| `invalid` | `boolean` | - | 输入框是否处于错误状态 |
| `disabled` | `boolean` | - | 是否禁用输入框 |
| `className` | `ClassNameValue` | - | 自定义 CSS 类，支持字符串、数组、对象、布尔值等多种格式。 |
| `ref` | `React.Ref<HTMLInputElement>` | - | 转发到输入元素的 ref |
