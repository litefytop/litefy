## API 参考

### ScrollShadow

带有渐变阴影指示器的可滚动容器。

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `React.ReactNode` | - | 可滚动内容 |
| `position` | `'top' \| 'bottom' \| 'left' \| 'right'` | `'bottom'` | 阴影指示器的位置 |
| `size` | `string` | `'64px'` | 阴影渐变的大小 |
| `slotProps` | `ScrollShadowSlotProps` | - | 用于自定义子元素的插槽属性 |
| `className` | `ClassNameValue` | - | 自定义 CSS 类。支持字符串、数组、对象、布尔值等。 |

### ScrollShadowSlotProps

| Slot | Type | Description |
|------|------|-------------|
| `shadow` | `React.ComponentProps<"div">` | 阴影覆盖层的属性 |
