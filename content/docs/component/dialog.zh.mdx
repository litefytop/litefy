## API 参考

### Dialog

使用原生 HTML dialog 元素的模态对话框组件。

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `ClassNameValue` | - | 自定义 CSS 类名，支持字符串、数组、对象、布尔值等多种格式。 |
| `onClose` | `() => void` | - | 对话框关闭时的回调函数 |
| `closeTrigger` | `boolean` | `true` | 是否显示 ESC 关闭按钮 |
| `ref` | `React.Ref<HTMLDialogElement>` | - | 转发到原生 dialog 元素的 ref |

### slotProps

| Slot | Type | Description |
|------|------|-------------|
| `closeTrigger` | `HTMLAttrs<Omit<React.ComponentProps<'button'>, 'onClick' \| 'type' \| 'aria-label'>>` | 关闭按钮的属性 |
