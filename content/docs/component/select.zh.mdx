## API 参考

### Select

一个样式化的选择器组件，支持受控/非受控状态、选项分组和验证。

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `className` | `ClassNameValue` | - | 自定义 CSS 类 |
| `invalid` | `boolean` | `false` | 将选择器标记为无效状态，用于表单验证 |
| `options` | `(SelectOption | SelectOptionGroup)[]` | - | 要显示的选项或选项组数组 |
| `placeholder` | `string` | - | 未选择任何选项时显示的占位符文本 |
| `value` | `string` | - | 受控模式下的选中值 |
| `defaultValue` | `string` | - | 非受控模式下的默认选中值 |
| `onChange` | `(e: React.ChangeEvent<HTMLSelectElement>) => void` | - | 选择变化时触发的回调函数 |

### SelectOption

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `label` | `string` | - | 选项的显示文本 |
| `value` | `string` | - | 选项关联的值 |

### SelectOptionGroup

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `group` | `string` | - | 分组标签文本 |
| `options` | `SelectOption[]` | - | 分组中的选项数组 |
