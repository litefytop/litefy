## API 参考

### Combobox

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | - | 受控值 |
| `defaultValue` | `string` | - | 默认值（非受控） |
| `onChange` | `(e: ComboboxChangeEvent) => void` | - | 值变化时的回调 |
| `onBlur` | `React.FocusEventHandler<HTMLInputElement>` | - | 输入框失焦时的回调 |
| `onSelect` | `(option: string) => void` | - | 选项被选中时的回调 |
| `options` | `string[] \| `<br/> `({page: number; size: number; keyword: string}) => Promise<{list: string[]; total: number}>` | - | 静态选项或带分页的异步获取函数 |
| `placeholder` | `string` | - | 输入框占位符 |
| `disabled` | `boolean` | `false` | 禁用组合框 |
| `invalid` | `boolean` | `false` | 标记为无效 |
| `debounceMs` | `number` | `300` | 异步搜索的防抖延迟 |
| `skeleton` | `React.ReactNode` | - | 自定义加载骨架 |
| `pageSize` | `number` | `20` | 分页每页加载的数据条数 |

### slotProps

| Slot | Type | Description |
|------|------|-------------|
| `container` | `HTMLAttrs<Omit<React.ComponentProps<"div">, "ref">>` | 容器 div 的属性 |
| `input` | `HTMLAttrs<Omit<React.ComponentProps<"input">, "disabled" \| "placeholder" \| "ref" \| "value" \| "defaultValue">>` | 输入框的属性 |
| `list` | `HTMLAttrs<Omit<React.ComponentProps<"ul">, "ref">>` | 下拉列表的属性 |
| `option` | `HTMLAttrs<React.ComponentProps<"li">>` | 每个选项的属性 |
