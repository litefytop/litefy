## API Reference

### Checkbox

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `React.ReactNode` | - | The checkbox content |
| `checked` | `boolean` | - | Controlled checked state |
| `onValueChange` | `(checked: boolean) => void` | - | Callback when checked state changes |
| `value` | `string` | - | Value for group mode |
| `variant` | `'checkbox' \| 'toggle'` | `'checkbox'` | Visual variant |
| `indicator` | `(checked: boolean, wrapperClassName?: string) => React.ReactNode` | - | Custom indicator render function with wrapper className |
| `disabled` | `boolean` | `false` | Disable the checkbox |
| `name` | `string` | - | Input name |
| `className` | `ClassNameValue` | - | Accepts strings, arrays, and all falsy values (false, null, undefined, 0, 0n). True and objects are not supported. |

### Checkbox.Group

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `React.ReactNode` | - | Checkbox items |
| `value` | `T[]` | - | Controlled value |
| `defaultValue` | `T[]` | `[]` | Default value (uncontrolled) |
| `onValueChange` | `(value: T[]) => void` | - | Callback when value changes |
| `disabled` | `boolean` | `false` | Disable all children |
| `invalid` | `boolean` | `false` | Mark as invalid |
| `name` | `string` | - | Group name |
| `className` | `ClassNameValue` | - | Accepts strings, arrays, and all falsy values (false, null, undefined, 0, 0n). True and objects are not supported. |
