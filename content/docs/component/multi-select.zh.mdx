## API 参考

### MultiSelect

支持选择多个选项的多选组件。

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `options` | `(SelectOption \| SelectOptionGroup)[]` | - | 选项列表或选项分组 |
| `value` | `string[]` | - | 受控的选中值 |
| `defaultValue` | `string[]` | `[]` | 非受控的默认选中值 |
| `onChange` | `(values: string[]) => void` | - | 选择变化时的回调函数 |
| `placeholder` | `string` | - | 占位文本 |
| `invalid` | `boolean` | `false` | 是否处于无效状态 |
| `className` | `ClassNameValue` | - | 自定义 CSS 类。支持字符串、数组、对象、布尔值等。 |

### SelectOption

| Prop | Type | Description |
|------|------|-------------|
| `label` | `string` | 选项的显示标签 |
| `value` | `string` | 选项的唯一值 |

### SelectOptionGroup

| Prop | Type | Description |
|------|------|-------------|
| `group` | `string` | 分组标签 |
| `options` | `SelectOption[]` | 该分组中的选项 |
