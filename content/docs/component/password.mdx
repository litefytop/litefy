## API Reference

### Password

A password input component with visibility toggle.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | - | Controlled password value |
| `invalid` | `boolean` | `false` | Whether the input is in invalid state |
| `direction` | `'vertical' \| 'horizontal'` | - | Layout direction (reserved for future use) |
| `slotProps` | `PasswordSlotProps` | - | Slot props for customizing sub-elements |
| `onChange` | `(e: React.ChangeEvent<HTMLInputElement>) => void` | - | Change event handler |
| `className` | `ClassNameValue` | - | Accepts strings, arrays, and all falsy values (false, null, undefined, 0, 0n). True and objects are not supported. |

### PasswordSlotProps

| Slot | Type | Description |
|------|------|-------------|
| `group` | `React.ComponentProps<"div">` | Props for the wrapper div element |
| `toggle` | `React.ComponentProps<"button">` | Props for the toggle button |
