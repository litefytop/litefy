## API Reference

### Upload

A styled file input component that supports all native file input attributes.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `invalid` | `boolean` | `false` | Marks the upload input as invalid for form validation |
| `accept` | `string` | - | Native file type filter (e.g. `image/*`, `.pdf,.doc`) |
| `multiple` | `boolean` | - | Allow selecting multiple files |
| `disabled` | `boolean` | - | Disable the file input |
| `name` | `string` | - | Input name for form submission |
| `onChange` | `(e: React.ChangeEvent<HTMLInputElement>) => void` | - | Callback triggered when files are selected |
| `className` | `ClassNameValue` | - | Custom CSS class. Supports strings, arrays, objects, booleans, and more. |
