## API 参考

### Radio

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `React.ReactNode` | - | 单选框内容 |
| `checked` | `boolean` | - | 受控选中状态 |
| `onValueChange` | `(checked: boolean) => void` | - | 选中状态变化时的回调 |
| `value` | `string` | - | 分组模式下的值 |
| `variant` | `'radio' \| 'segment'` | `'radio'` | 视觉变体 |
| `indicator` | `(checked: boolean, wrapperClassName?: string) => React.ReactNode` | - | 自定义指示器渲染函数，包含包装器className |
| `disabled` | `boolean` | `false` | 禁用单选框 |
| `name` | `string` | - | 输入框名称 |
| `invalid` | `boolean` | `false` | 标记为无效 |
| `className` | `ClassNameValue` | - | 自定义 CSS 类。仅支持字符串、数组及各类假值。 |

### Radio.Group

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `React.ReactNode` | - | 单选框子项 |
| `value` | `T` | - | 受控值 |
| `defaultValue` | `T` | - | 默认值（非受控） |
| `onValueChange` | `(value: T) => void` | - | 值变化时的回调 |
| `disabled` | `boolean` | `false` | 禁用所有子项 |
| `invalid` | `boolean` | `false` | 标记为无效 |
| `name` | `string` | - | 分组名称 |
| `variant` | `'radio' \| 'segment'` | `'radio'` | 所有子项的视觉变体 |
| `className` | `ClassNameValue` | - | 自定义 CSS 类。仅支持字符串、数组及各类假值。 |
