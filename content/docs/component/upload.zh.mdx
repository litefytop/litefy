## API 参考

### Upload

支持所有原生文件输入属性的样式化上传组件。

| 属性 | 类型 | 默认值 | 描述 |
|------|------|---------|-------------|
| `invalid` | `boolean` | `false` | 将上传输入标记为无效状态，用于表单验证 |
| `accept` | `string` | - | 原生文件类型过滤（例如 `image/*`、`.pdf,.doc`） |
| `multiple` | `boolean` | - | 允许选择多个文件 |
| `disabled` | `boolean` | - | 禁用文件输入 |
| `name` | `string` | - | 用于表单提交的输入名称 |
| `onChange` | `(e: React.ChangeEvent<HTMLInputElement>) => void` | - | 选择文件后触发的回调函数 |
| `className` | `ClassNameValue` | - | 自定义 CSS 类。仅支持字符串、数组及各类假值。 |
