## API Reference

### ScrollShadow

A scrollable container with gradient shadow indicators.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `React.ReactNode` | - | Scrollable content |
| `position` | `'top' \| 'bottom' \| 'left' \| 'right'` | `'bottom'` | Position of the shadow indicator |
| `size` | `string` | `'64px'` | Size of the shadow gradient |
| `slotProps` | `ScrollShadowSlotProps` | - | Slot props for customizing sub-elements |
| `className` | `ClassNameValue` | - | Accepts strings, arrays, and all falsy values (false, null, undefined, 0, 0n). True and objects are not supported. |

### ScrollShadowSlotProps

| Slot | Type | Description |
|------|------|-------------|
| `shadow` | `React.ComponentProps<"div">` | Props for the shadow overlay element |
