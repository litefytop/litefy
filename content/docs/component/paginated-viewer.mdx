## API Reference

### PaginatedViewer

The main container component that displays one child at a time based on the active index.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `React.ReactNode` | - | Slide elements to paginate through |
| `activeIndex` | `number` | - | Index of the currently visible slide |
| `className` | `ClassNameValue` | - | Accepts strings, arrays, and all falsy values (false, null, undefined, 0, 0n). True and objects are not supported. |

### slotProps

| Slot | Type | Description |
|------|------|-------------|
| `slide` | `React.ComponentProps<"div">` | Props for the slide wrapper div |
