## API 参考

### Password

支持可见性切换的密码输入组件。

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | - | 受控的密码值 |
| `invalid` | `boolean` | `false` | 是否处于无效状态 |
| `direction` | `'vertical' \| 'horizontal'` | - | 布局方向（保留供未来使用） |
| `slotProps` | `PasswordSlotProps` | - | 用于自定义子元素的插槽属性 |
| `onChange` | `(e: React.ChangeEvent<HTMLInputElement>) => void` | - | 变更事件处理函数 |
| `className` | `ClassNameValue` | - | 自定义 CSS 类。支持字符串、数组、对象、布尔值等。 |

### PasswordSlotProps

| Slot | Type | Description |
|------|------|-------------|
| `group` | `React.ComponentProps<"div">` | 包装 div 元素的属性 |
| `toggle` | `React.ComponentProps<"button">` | 切换按钮的属性 |
