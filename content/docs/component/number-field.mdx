## API Reference

### NumberField

A numeric input field with built-in increment and decrement buttons.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `defaultValue` | `number \| string` | - | Initial value (uncontrolled mode) |
| `value` | `number \| string \| null` | - | Current value (controlled mode) |
| `onValueChange` | `(value: number \| null) => void \| (value: string) => void` | - | Value change handler. Returns `number \| null` when `positiveInteger=true`, otherwise `string` |
| `className` | `ClassNameValue` | - | Accepts strings, arrays, and all falsy values (false, null, undefined, 0, 0n). True and objects are not supported. |
| `invalid` | `boolean` | `false` | Whether the field is in an invalid state |
| `min` | `number` | `0` (positiveInteger) or `-Infinity` | Minimum allowed value |
| `max` | `number` | `Infinity` | Maximum allowed value |
| `step` | `number` | `1` | Increment/decrement step size |
| `positiveInteger` | `boolean` | `false` | Restrict to positive integers only. Changes value type to `number \| null` |
| `disabled` | `boolean` | `false` | Disable the entire field |
| `onBlur` | `React.FocusEventHandler<HTMLInputElement>` | - | Blur event handler |

### slotProps

| Slot | Type | Description |
|------|------|-------------|
| `group` | `React.ComponentProps<"div">` | Props for the wrapper div |
| `step` | `React.ComponentProps<"button">` | Props for increment/decrement buttons |
