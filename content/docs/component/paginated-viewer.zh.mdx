## API 参考

### PaginatedViewer

根据活动索引一次显示一个子元素的主要容器组件。

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `React.ReactNode` | - | 要分页的幻灯片元素 |
| `activeIndex` | `number` | - | 当前可见幻灯片的索引 |
| `className` | `ClassNameValue` | - | 自定义 CSS 类。支持字符串、数组、对象、布尔值等。 |

### slotProps

| Slot | Type | Description |
|------|------|-------------|
| `slide` | `React.ComponentProps<"div">` | 幻灯片包装 div 的属性 |
