## API 参考

### Tabs

标签页的主要容器组件。

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `defaultValue` | `string` | - | 初始活动标签值（非受控模式） |
| `value` | `string` | - | 活动标签值（受控模式） |
| `onValueChange` | `(value: string) => void` | - | 活动标签变化时的回调 |
| `orientation` | `'horizontal' \| 'vertical'` | `'horizontal'` | 布局方向 |
| `variant` | `'button' \| 'line'` | `'line'` | 视觉样式变体 |
| `className` | `ClassNameValue` | - | 自定义 CSS 类。支持字符串、数组、对象、布尔值等。 |

### Tab

单个标签面板组件。

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | - | 标签的唯一标识符 |
| `label` | `React.ReactNode` | - | 标签触发器的标签内容 |
| `disabled` | `boolean` | `false` | 禁用该标签 |
| `className` | `ClassNameValue` | - | 自定义 CSS 类。支持字符串、数组、对象、布尔值等。 |

### slotProps

| Slot | Type | Description |
|------|------|-------------|
| `list` | `React.ComponentProps<"div">` | 标签列表容器的属性 |
| `trigger` | `React.ComponentProps<"button">` | 标签触发按钮的属性 |
