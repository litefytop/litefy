## API 参考

### NumberField

带有内置增减按钮的数字输入字段。

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `defaultValue` | `number \| string` | - | 初始值（非受控模式） |
| `value` | `number \| string \| null` | - | 当前值（受控模式） |
| `onValueChange` | `(value: number \| null) => void \| (value: string) => void` | - | 值变化处理器。当 `positiveInteger=true` 时返回 `number \| null`，否则返回 `string` |
| `className` | `ClassNameValue` | - | 自定义 CSS 类。仅支持字符串、数组及各类假值。 |
| `invalid` | `boolean` | `false` | 是否处于无效状态 |
| `min` | `number` | `0`（positiveInteger）或 `-Infinity` | 允许的最小值 |
| `max` | `number` | `Infinity` | 允许的最大值 |
| `step` | `number` | `1` | 增减步长 |
| `positiveInteger` | `boolean` | `false` | 限制为正整数。将值类型更改为 `number \| null` |
| `disabled` | `boolean` | `false` | 禁用整个字段 |
| `onBlur` | `React.FocusEventHandler<HTMLInputElement>` | - | 失去焦点事件处理器 |

### slotProps

| Slot | Type | Description |
|------|------|-------------|
| `group` | `React.ComponentProps<"div">` | 包装 div 的属性 |
| `step` | `React.ComponentProps<"button">` | 增减按钮的属性 |
