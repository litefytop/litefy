## API 参考

### Drawer

带有焦点管理和键盘导航的滑入面板组件。

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `ClassNameValue` | - | 自定义 CSS 类名，支持字符串、数组、对象、布尔值等多种格式。 |
| `placement` | `'left' \| 'right' \| 'top' \| 'bottom'` | `'right'` | 抽屉的位置 |
| `onClose` | `() => void` | - | 抽屉关闭时的回调函数 |
| `ref` | `React.Ref<HTMLDialogElement>` | - | 转发到原生 dialog 元素的 ref |

### slotProps

| Slot | Type | Description |
|------|------|-------------|
| `content` | `HTMLAttrs<Omit<React.ComponentProps<'div'>, 'style'>>` | 内容面板的属性 |
