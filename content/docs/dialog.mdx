## API Reference

### Dialog

A modal dialog component that uses the native HTML dialog element.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `ClassNameValue` | - | Custom CSS class. Supports strings, arrays, objects, booleans, and more. |
| `onClose` | `() => void` | - | Callback when dialog is closed |
| `closeTrigger` | `boolean` | `true` | Whether to show the ESC close button |
| `ref` | `React.Ref<HTMLDialogElement>` | - | Ref forwarded to the native dialog element |

### slotProps

| Slot | Type | Description |
|------|------|-------------|
| `closeTrigger` | `HTMLAttrs<Omit<React.ComponentProps<'button'>, 'onClick' \| 'type' \| 'aria-label'>>` | Props for the close button |
