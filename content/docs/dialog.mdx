## API Reference

### Dialog

A modal dialog component that uses the native HTML dialog element.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `ClassNameValue` | `undefined` | Custom CSS class |
| `onClose` | `() => void` | `undefined` | Callback when dialog is closed |

### DialogRef

The ref object returned by the Dialog component.

| Method | Description |
|--------|-------------|
| `showModal()` | Open the dialog |
| `close()` | Close the dialog |
