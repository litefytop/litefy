## API Reference

### Accordion

The main container component.



| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `allowMultiple` | `boolean` | `false` | Enable multiple items to be open simultaneously |
| `defaultOpenKeys` | `string[]` | `[]` | Default open item values (uncontrolled) |
| `openKeys` | `string[]` | `undefined` | Open item values (controlled) |
| `onOpenChange` | `(keys: string[]) => void` | `undefined` | Callback when open keys change |
| `className` | `string` | `undefined` | Custom CSS class for the container |

### Accordion.Item

Individual accordion item.



| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | - | Unique identifier for the item |
| `label` | `React.ReactNode` | - | The header content |
| `icon` | `React.ReactNode \| ((open: boolean) => React.ReactNode)` | `ChevronDown` | Custom icon or icon function |
| `disabled` | `boolean` | `false` | Disable the item |
| `className` | `string` | `undefined` | Custom CSS class |
| `slotProps` | `object` | `undefined` | Custom props for internal slots |

### slotProps

| Slot | Type | Description |
|------|------|-------------|
| `wrapper` | `React.HTMLAttributes<HTMLDivElement>` | Props for the wrapper div |
| `trigger` | `React.ButtonHTMLAttributes<HTMLButtonElement>` | Props for the trigger button |
| `content` | `React.HTMLAttributes<HTMLDivElement>` | Props for the content panel |
