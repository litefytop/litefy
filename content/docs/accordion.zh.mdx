## API 参考

### Accordion

主容器组件。



| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `allowMultiple` | `boolean` | `false` | 启用多个项目同时展开 |
| `defaultOpenKeys` | `string[]` | `[]` | 默认展开项的值（非受控） |
| `openKeys` | `string[]` | `undefined` | 展开项的值（受控） |
| `onOpenChange` | `(keys: string[]) => void` | `undefined` | 展开键变化时的回调 |
| `className` | `string` | `undefined` | 容器的自定义 CSS 类 |

### Accordion.Item

单个手风琴项。



| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | - | 项目的唯一标识符 |
| `label` | `React.ReactNode` | - | 头部内容 |
| `icon` | `React.ReactNode \| ((open: boolean) => React.ReactNode)` | `ChevronDown` | 自定义图标或图标函数 |
| `disabled` | `boolean` | `false` | 禁用项目 |
| `className` | `string` | `undefined` | 自定义 CSS 类 |
| `slotProps` | `object` | `undefined` | 内部插槽的自定义属性 |

### slotProps

| Slot | Type | Description |
|------|------|-------------|
| `wrapper` | `React.ComponentProps<"div">` | 包装器 div 的属性 |
| `trigger` | `React.ComponentProps<"button">` | 触发器按钮的属性 |
| `content` | `React.ComponentProps<"section">` | 内容面板的属性 |
