## API 参考

### Dialog

使用原生 HTML dialog 元素的模态对话框组件。

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `ClassNameValue` | `undefined` | 自定义 CSS 类名 |
| `onClose` | `() => void` | `undefined` | 对话框关闭时的回调函数 |

### DialogRef

Dialog 组件返回的 ref 对象。

| Method | Description |
|--------|-------------|
| `showModal()` | 打开对话框 |
| `close()` | 关闭对话框 |
