## 介绍

Litefy UI 是一个几乎不依赖外部库的轻量化组件库。每个组件都支持复制粘贴使用，并且提供了 CLI 来加速接入。

## 安装

<Tabs items={['CLI', '手动']}>

<Tab value="CLI">

最快的接入方式是通过 CLI。它会直接把源码添加到你的项目中，方便你自由修改。

<PackageManagerTabs command="litefy add button" />

</Tab>

<Tab value="手动">

你也可以直接从文档页复制组件源码并粘贴到项目中。每个组件页都展示了完整源码。

<SourceCode component="button" />

</Tab>

</Tabs>

## 下一步

- 在侧边栏的 **组件** 分类中浏览并添加更多组件。
- 打开 **总览** 页面，快速查看所有组件、CSS 工具类与钩子的索引。
- 前往 **样式** 分类查看 `Card`、`Kbd`、`Skeleton` 等工具类。
