## API 参考

### Button

按钮组件。

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `'primary' \| 'destructive' \| 'outline' \| 'text'` | `'primary'` | 按钮视觉变体 |
| `className` | `ClassNameValue` | `undefined` | 自定义 CSS 类 |
| `loadingConfig` | `ButtonLoadingConfig` | `undefined` | 加载状态配置 |

### ButtonLoadingConfig

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `loading` | `boolean` | `false` | 按钮是否处于加载状态 |
| `icon` | `React.ReactNode` | `undefined` | 自定义加载图标 |
