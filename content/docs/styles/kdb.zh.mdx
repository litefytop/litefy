---
title: 键盘按键
description: 用于渲染键盘快捷键和单个按键标签的键盘按键样式。
---

## 源码

```css
.kdb {
  display: inline-flex;
  align-items: center;
  justify-content: center;
  padding-inline: 0.625rem;
  padding-block: 0.375rem;
  min-width: 2.4rem;
  height: 2rem;
  font-size: 0.75rem;
  font-family: monospace;
  font-weight: 600;
  letter-spacing: 0.025em;
  color: var(--foreground);
  background-color: var(--kbd-bg);
  backdrop-filter: blur(8px);
  border: 1px solid var(--kbd-border);
  border-radius: var(--radius);
  box-shadow: 0 1px 2px var(--kbd-shadow);
  transition: all 200ms cubic-bezier(0.4, 0, 0.2, 1);
}

.kdb:hover {
  background-color: var(--kbd-bg-hover);
  border-color: var(--kbd-border-hover);
  box-shadow: 0 4px 6px -1px var(--kbd-shadow-hover);
  transform: translateY(-1px);
}

.kdb:active {
  background-color: var(--kbd-bg-active);
  border-color: var(--kbd-border-active);
  box-shadow: inset 0 2px 4px var(--kbd-shadow-active);
  transform: translateY(1px);
}
```
