## API 参考

### Image

带有加载状态管理的图片组件。

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `src` | `string` | - | 图片源 URL |
| `alt` | `string` | - | 图片的替代文本 |
| `className` | `ClassNameValue` | - | 自定义 CSS 类，支持字符串、数组、对象、布尔值等多种格式。 |
| `fallback` | `React.ReactNode` | - | 图片加载失败时显示的自定义回退元素 |
| `placeholderSrc` | `string` | - | 加载期间显示的占位图片 URL |
