# Separator

A visual divider component for separating content.

## Installation

```tsx
import { Separator } from "@/component";
```

## Usage

```tsx
<Separator />
```

## API Reference

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| orientation | "horizontal" \| "vertical" | "horizontal" | The orientation of the separator |
| children | ReactNode | - | Text content to display in the center |
| className | ClassNameValue | - | Container className |
| lineClassName | ClassNameValue | - | Line element className |
