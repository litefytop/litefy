## API 参考

### Input

带有插槽和验证支持的输入框组件。

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `type` | `'text' \| 'email' \| 'url' \| 'tel' \| 'search'` | `'text'` | 输入类型 |
| `className` | `ClassNameValue` | - | 自定义 CSS 类，支持字符串、数组、对象、布尔值等多种格式。 |
| `leading` | `ReactNode` | - | 前置插槽内容（如图标） |
| `trailing` | `ReactNode` | - | 后置插槽内容（如按钮、图标） |
| `invalid` | `boolean` | `false` | 输入框是否处于无效状态 |
| `slotProps` | `InputSlotProps` | - | 用于自定义的插槽属性 |

### InputSlotProps

| Slot | Type | Description |
|------|------|-------------|
| `group` | `HTMLAttributes<HTMLDivElement>` | 输入框组容器的属性 |
| `leading` | `HTMLAttributes<HTMLSpanElement>` | 前置插槽容器的属性 |
| `trailing` | `HTMLAttributes<HTMLSpanElement>` | 后置插槽容器的属性 |
