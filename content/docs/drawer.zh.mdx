## API 参考

### Drawer

带有焦点管理和键盘导航的滑入面板组件。

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `ClassNameValue` | `undefined` | 自定义 CSS 类名 |
| `placement` | `'left' \| 'right' \| 'top' \| 'bottom'` | `'right'` | 抽屉的位置 |
| `onClose` | `() => void` | `undefined` | 抽屉关闭时的回调函数 |
| `onOpen` | `() => void` | `undefined` | 抽屉打开时的回调函数 |
| `aria-label` | `string` | `undefined` | 无障碍标签 |
| `aria-labelledby` | `string` | `undefined` | 标记抽屉的元素 ID |

### DrawerRef

Drawer 组件返回的 ref 对象。

| Method | Description |
|--------|-------------|
| `show()` | 打开抽屉 |
| `close()` | 关闭抽屉 |
