## API Reference

### Input

The input component with slot and validation support.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `type` | `'text' \| 'email' \| 'url' \| 'tel' \| 'search'` | `'text'` | Input type |
| `className` | `ClassNameValue` | - | Custom CSS class. Supports strings, arrays, objects, booleans, and more. |
| `leading` | `ReactNode` | - | Leading slot content (e.g., icons) |
| `trailing` | `ReactNode` | - | Trailing slot content (e.g., buttons, icons) |
| `invalid` | `boolean` | `false` | Whether the input is in an invalid state |
| `slotProps` | `InputSlotProps` | - | Slot props for customization |

### InputSlotProps

| Slot | Type | Description |
|------|------|-------------|
| `group` | `HTMLAttributes<HTMLDivElement>` | Props for the input group container |
| `leading` | `HTMLAttributes<HTMLSpanElement>` | Props for the leading slot container |
| `trailing` | `HTMLAttributes<HTMLSpanElement>` | Props for the trailing slot container |
