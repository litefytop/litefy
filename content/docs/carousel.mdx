## API Reference

### Carousel

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `React.ReactNode` | - | Slides content |
| `activeIndex` | `number` | - | Current active slide index |
| `onChange` | `(index: number) => void` | `undefined` | Callback when slide changes |
| `autoPlay` | `boolean` | `false` | Enable automatic slide rotation |
| `autoPlayInterval` | `number` | `3000` | Interval between slides in milliseconds |
| `loop` | `boolean` | `false` | Enable infinite loop |
| `slotProps` | `object` | `undefined` | Custom props for internal slots |

### slotProps

| Slot | Type | Description |
|------|------|-------------|
| `wrapper` | `React.ComponentProps<"section">` | Props for the wrapper div |
| `list` | `React.ComponentProps<"ul">` | Props for the list element |
| `slide` | `React.ComponentProps<"li">` | Props for each slide div |
