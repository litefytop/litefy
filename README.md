# Litefy UI
A lightweight, fully transparent React component library built with TailwindCSS.
All component source code is delivered directly to your project, no black-box encapsulation. You own the code, control your UI entirely.

## Documentation
Visit official site: https://litefy.top

## Core Concepts
- Open Source Code: Directly edit component source files without hidden layers
- Composition First: Unified component API for consistent development experience
- Lightweight Distribution: Install components via CLI, free from third-party package version risks
- Native Web First: Prefer browser native standards, reduce redundant compatibility code
