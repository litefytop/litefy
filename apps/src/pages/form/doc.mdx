# Form

Form component for handling form submissions with validation support.

## Installation

```tsx
import { Form } from "@/component";
```

## Usage

### Basic Form

A simple uncontrolled form example.

```tsx
import { Form, Input, Button } from "@/component";

export default function BasicForm() {
  const handleSubmit = async (formData: FormData) => {
    const data = Object.fromEntries(formData);
    console.log("Form submitted:", data);
    return true;
  };

  return (
    <Form onSubmit={handleSubmit}>
      <Form.Field label="Username" name="username">
        {(field) => <Input {...field} placeholder="Enter username" />}
      </Form.Field>
      <Form.Submit>Submit</Form.Submit>
    </Form>
  );
}
```

## API

### Form Props

| Property | Type | Default | Description |
| --- | --- | --- | --- |
| `onSubmit` | `(formData: FormData) => Promise<boolean>` | - | Form submit handler |

### Form.Field Props

| Property | Type | Default | Description |
| --- | --- | --- | --- |
| `label` | `ReactNode` | - | Field label |
| `name` | `string` | - | Field name (required) |
| `description` | `ReactNode` | - | Field description |
| `invalid` | `ReactNode` | - | Invalid message |
| `disabled` | `boolean` | `false` | Whether disabled |
| `direction` | `"vertical" \| "horizontal"` | `"vertical"` | Layout direction |

### Form.Submit Props

| Property | Type | Default | Description |
| --- | --- | --- | --- |
| `loading` | `boolean` | `false` | Loading state |
