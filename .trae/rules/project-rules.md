# 项目规则

## 项目结构

```
src/
├── assets/           # 静态资源
│   ├── img/          # 图片
│   ├── font/         # 字体
│   └── style/        # 样式 CSS 文件
├── component/        # 项目专用组件
│   └── ui/           # 项目通用 UI 组件
├── lib/              # 工具函数和第三方库封装
└── pages/            # 页面（用户所看到的）
    └── config/       # 路由和国际化配置
```

## 文件命名

- 文件命名始终使用 Kabab-Case
