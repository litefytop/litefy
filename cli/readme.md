

# litefy

A minimal React UI library CLI – add components directly to your project.

```bash
npx litefy init
npx litefy add button
```

## Features

- 🎨 **Zero runtime dependencies** (only `lucide-react` for icons)
- 📦 **Copy, don't install** – source code goes into your project, fully editable
- 🌍 **Global CDN** – components served via jsDelivr
- 🔒 **Secure** – no external registry requests after CLI install
- 📄 **Optional documentation** – download markdown docs with `--docs`

## Installation

No installation required – use directly with `npx`:

```bash
npx litefy init
```

Or install globally:

```bash
npm install -g litefy
litefy init
```

## Usage

### Initialize configuration

```bash
litefy init
```

This creates a `litefy.json` file in your project root.

Options:
- `-y, --yes` – skip prompts, use defaults

### Add components

```bash
litefy add button
litefy add modal drawer
litefy add accordion --docs
```

Options:
- `-o, --overwrite` – overwrite existing files
- `-d, --docs` – also download markdown documentation into `docs/`

### Available components

All components are listed in the [registry](https://github.com/litefytop/litefy/blob/main/apps/public/registry.json). Currently:

`accordion`, `anchor`, `button`, `carousel`, `checkbox`, `combobox`, `date-picker`, `description`, `dialog`, `drawer`, `dropdown`, `empty`, `form`, `image`, `input`, `loading`, `number-field`, `paginated-viewer`, `pagination`, `paper`, `password`, `progress`, `radio`, `select`, `separator`, `sidebar`, `slider`, `tabs`, `text-area`, `theme`, `title`, `toast`, `tooltip`, `upload`, `virtual-scroll`, `watermark`.

## Configuration

`litefy.json` (generated by `litefy init`):

```json
{
  "components": "./src/components",
  "installed": ["button", "modal"]
}
```

- `components` – target directory for component source files
- `installed` – list of already installed components (managed automatically)

## How it works

1. `litefy init` creates a configuration file.
2. `litefy add <component>` fetches the component registry from `https://cdn.jsdelivr.net/gh/litefytop/litefy@main/apps/public/registry.json`.
3. The registry points to CDN URLs with a fixed version tag (e.g. `@v0.0.1`).
4. CLI downloads the raw `.tsx` (and optional `.md`) files into your project.
5. You own the code – change anything, no hidden dependencies.

## Versioning

The CLI itself is versioned on npm. The component registry uses Git tags from the [litefy GitHub repo](https://github.com/litefytop/litefy/releases). To use a specific version of the components, install the corresponding CLI version:

```bash
npx litefy@0.0.1 add button
```
