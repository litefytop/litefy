## API Reference

### Radio.Group

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| defaultValue | T | - | Default value for uncontrolled mode |
| value | T | - | Controlled value |
| onValueChange | (value: T) => void | - | Callback when value changes |
| direction | horizontal \| vertical | horizontal | Layout direction |
| disabled | boolean | false | Disable the entire group |
| invalid | boolean | false | Set invalid state |
| className | ClassNameValue | - | Additional class name |

### Radio

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| value | string | required | Radio value |
| variant | primary \| secondary \| outlined \| text | text | Style variant |
| disabled | boolean | false | Disable this radio item |
| indicator | object | - | Custom indicator configuration |
| indicator.checked | ReactNode | - | Custom checked indicator |
| indicator.unchecked | ReactNode | - | Custom unchecked indicator |
| indicator.hidden | boolean | false | Hide indicator |
| indicator.className | ClassNameValue | - | Indicator class name |
| indicator.props | ComponentProps\<span\> | - | Span props for indicator |
| onValueChange | (value: string) => void | - | Callback when this radio is selected |
| className | ClassNameValue | - | Additional class name |
