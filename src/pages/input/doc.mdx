## 用法

```tsx
import { Input } from "@/component/ui/input";

function App() {
  return <Input placeholder="Enter text..." />;
}
```

## 示例

### 基础用法

```tsx
<Input placeholder="Enter text..." />
```

### 前缀后缀

```tsx
<Input prefix="https://" suffix=".com" />
<Input prefix="$" suffix=".00" />
```

### 错误状态

```tsx
<Input invalid error="This field is required" />
```

### 禁用状态

```tsx
<Input disabled placeholder="Disabled input" />
```

## API Reference

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| invalid | boolean | - | 是否无效状态 |
| label | ReactNode | - | 标签 |
| description | ReactNode | - | 描述信息 |
| error | ReactNode | - | 错误信息 |
| prefix | ReactNode | - | 前缀内容 |
| suffix | ReactNode | - | 后缀内容 |
| itemProps | { label?, group?, prefix?, suffix?, error?, description? } | - | 各部分属性 |
| ...HTMLInputElementProps | HTMLInputElementProps | - | 继承 input 元素的其他 props |
