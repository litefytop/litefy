## CSS Classes

```css
// CheckboxGroup Base
base: "w-full flex gap-2"

// Direction
horizontal: "flex-wrap"
vertical: "flex-col"

// Checkbox Item
item: "flex items-center h-9 px-2 gap-2 cursor-pointer 
      focus-visible:[&_svg]:fill-ring 
      aria-invalid:[&_svg]:stroke-destructive/20 
      data-[state=on]:[&_svg]:fill-primary data-[state=on]:[&_svg]:stroke-primary-foreground 
      disabled:opacity-50 disabled:text-muted-foreground 
      disabled:[&_svg]:fill-muted disabled:[&_svg]:stroke-muted"
```

## API Reference

### Checkbox.Group Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| defaultValue | T[] | - | 初始选中的值（非受控模式） |
| value | T[] | - | 当前选中的值（受控模式） |
| onValueChange | (value: T[]) => void | - | 值变化回调 |
| direction | "horizontal" \| "vertical" | "horizontal" | 排列方向 |
| disabled | boolean | false | 是否禁用整组 |
| className | string | - | 自定义类名 |

### Checkbox Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| value | string | - | 唯一标识符（必填） |
| disabled | boolean | false | 是否禁用 |
| children | ReactNode | - | 显示文本 |
