```tsx

function App() {
```

## API Reference

### Anchor Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| children | ReactNode | - | 子组件 |
| className | string | - | 自定义类名 |

### Anchor.Item Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| href | string | - | 锚点目标 ID（带 # 前缀） |
| children | ReactNode | - | 链接文本 |
| className | string | - | 自定义类名 |
| onClick | () => void | - | 点击回调 |

### Anchor.Section Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| href | string | - | 锚点目标 ID（带 # 前缀，可选） |
| title | string | - | 章节标题 |
| children | ReactNode | - | 子锚点项 |
| className | string | - | 自定义类名 |
| itemProps | object | - | 内部封装属性透传 |

### itemProps 配置

| 属性 | 类型 | 描述 |
|------|------|------|
| title | Omit<React.ComponentProps<"a">, "href" \| "onClick"> | 标题 `<a>` 标签属性 |
| nav | Omit<React.ComponentProps<"nav">, "aria-label"> | 内部导航 `<nav>` 容器属性 |
