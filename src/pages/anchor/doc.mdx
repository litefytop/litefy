## API Reference

### Anchor Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| children | ReactNode | - | 子组件 |
| className | string | - | 自定义类名 |

### Anchor.Item Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| id | string | - | 锚点目标 ID（带 # 前缀） |
| children | ReactNode | - | 链接文本 |
| className | string | - | 自定义类名 |
| onClick | () => void | - | 点击回调 |

### Anchor.Section Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| id | string | - | 锚点目标 ID |
| title | string | - | 章节标题 |
| children | ReactNode | - | 子锚点项 |
| className | string | - | 自定义类名 |
| itemProps | object | - | 子元素属性透传配置 |

### itemProps 配置

| 属性 | 类型 | 描述 |
|------|------|------|
| title | Omit<React.ComponentProps<"a">, "href" \| "onClick"> | 标题链接属性 |
| nav | Omit<React.ComponentProps<"nav">, "aria-label"> | 导航容器属性 |
