# Litefy UI
一套基于 TailwindCSS 构建、轻量化、全透明的 React 组件库。
所有组件源码直接交付项目，无黑盒封装，代码完全归你掌控，自主定制页面样式与逻辑。

## 文档站点
访问官网：https://litefy.top/zh

## 核心理念
- 源码开放：直接修改组件源文件，不存在封闭封装层
- 组合优先：统一组件设计 API，降低学习成本
- 轻量分发：CLI 本地安装组件，不受第三方包迭代 Breaking Change 影响
- 原生优先：优先使用浏览器标准原生能力，精简冗余兼容代码
