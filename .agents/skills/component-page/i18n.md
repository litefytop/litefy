# i18n 翻译配置说明

## 通用翻译 (common)

所有组件通用的翻译，放在 `common` 下：

```ts
// zh.ts
common: {
  copyDocs: "复制文档",
  copySuccess: "复制成功",
  className: "支持 string | string[] | boolean | null | undefined | Record<string,string | string[]>，用于添加自定义样式",
}

// en.ts
common: {
  copyDocs: "Copy docs",
  copySuccess: "Copied!",
  className: "Supports string | string[] | boolean | null | undefined | Record<string,string | string[]>, used to add custom styles",
}
```

## 组件翻译结构

每个组件的翻译结构：

```ts
componentName: {
  title: "组件名",
  description: "组件描述...",
  
  // 示例章节（根据需要添加）
  basic: {
    title: "基础",
  },
  advanced: {
    title: "高级",
  },
  
  // 结构章节
  anatomy: {
    root: "根元素",
    child: "子元素",
  },
  
  // API 文档
  api: {
    // 章节标题
    sectionTitles: {
      componentProps: "ComponentName Props",
    },
    
    // 属性描述
    props: {
      propName: "属性描述",
    },
    
    // 列标题（可选，默认有默认值）
    headers: {
      prop: "Prop",
      type: "Type", 
      default: "Default",
      description: "描述",
    },
  },
},
```

## 使用方式

在 page.tsx 中：

```tsx
export default function ComponentPage({ locale = "zh" }: { locale?: string }) {
  const lang = t(locale as "zh" | "en");  // 完整 i18n 对象
  const l = lang.componentName;            // 当前组件翻译
  
  // 使用
  l.title                    // 组件标题
  l.description             // 组件描述
  l.basic.title             // 示例标题
  l.api.props.propName      // 属性描述
  lang.common.className     // 通用 className 描述
}
```

## 添加新组件翻译步骤

1. 在 `zh.ts` 和 `en.ts` 中添加组件翻译
2. 在 page.tsx 中使用 `sections` 数组定义 API 文档
3. 使用 `l.api.props.*` 获取属性描述
4. className 使用 `lang.common.className`
